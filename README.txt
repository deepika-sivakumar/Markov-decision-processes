Markov Decision Processes

1. Download/clone the code from the repository - 
	https://github.com/deepika-sivakumar/cs7641-mdp.git
2. Python version used - python 3.8
3. Packages to install 
	lrl
	pymdptoolbox
	pandas
	numpy
	matplotlib
4. Steps to run the code:
	a. Install Anaconda: https://docs.anaconda.com/anaconda/install/
	b. Create conda environment: conda env create -f env.yml
	c. Activate the environment: conda activate ml
	d. Run the Jupyter Server: jupyter notebook
	e. In the web browser that opens, Open the following ipynb files and run all the cells to generate graphs
		rl_frozen_lake.ipynb
		rl_forest.ipynb
5. The graphs and visualization figures will be generated in the notebook.
